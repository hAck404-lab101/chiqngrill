# Feature: Luxury Homepage

## Purpose

Create a premium first impression for Chiq-N-Grill and convert visitors into customers through order, call, menu, and directions CTAs.

## User Flow

1. Visitor lands on homepage.
2. Visitor sees hero image/video and main tagline.
3. Visitor chooses one of the primary actions:
   - Order now
   - View menu
   - Get directions
   - Call restaurant
4. Visitor scrolls through favorites, vibe, reviews, and location.
5. Visitor sees final CTA and takes action.

## Required Sections

- Sticky header
- Hero
- Trust badges
- Customer-loved favorites
- Order CTA strip
- Experience section
- Gallery
- Review highlights
- Location section
- Final CTA
- Footer

## Main Copy

Headline:

`Bold Chicken. Smooth Vibes. Accra’s Grill Experience.`

Subtext:

`Enjoy well-seasoned chicken, comforting rice meals, fries, drinks, and a relaxed dine-in atmosphere on Papa Monrovia Street, Accra.`

## Design Notes

- Use dark premium background.
- Use warm food photography.
- Avoid generic food website layout.
- Use custom Chiq-N-Grill content.
- Keep mobile CTAs sticky and visible.
