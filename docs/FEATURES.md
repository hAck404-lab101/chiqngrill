# Feature Roadmap

## MVP Features

### 1. Luxury Homepage

Purpose: introduce the brand, sell the food visually, and push users to order/call/get directions.

Required sections:

- Sticky header
- Hero section
- Popular menu preview
- Order CTA strip
- Experience/vibe section
- Gallery
- Reviews
- Location map
- Final CTA
- Footer

### 2. Digital Menu

Required:

- Menu categories
- Food cards
- Price display
- Short descriptions
- Spice level
- Prep time
- Add-to-cart CTA
- Popular / Chef’s Pick / Customer Favorite badges

### 3. Cart & Checkout Flow

Required:

- Add item to cart
- Update quantity
- Select order type: dine-in, pickup, delivery, kerbside pickup
- Add customer name and phone number
- Add delivery address when delivery is selected
- Add order notes
- Confirm order
- Generate order reference

### 4. WhatsApp / Call Fallback

Required:

- Floating call button
- Floating WhatsApp button
- Pre-filled WhatsApp order summary
- Click-to-call phone link

### 5. Location & Local SEO

Required:

- Papa Monrovia Street, Accra
- Plus code: HR78+C3 Accra
- Google Maps embed placeholder
- Directions CTA
- Local SEO copy
- Restaurant schema planning

### 6. Admin Dashboard Planning

Required:

- Menu management
- Order management
- Review management
- Gallery management
- Promo codes
- Opening hours
- Delivery zones

## Feature Documentation Rule

Every feature must have a Markdown reference file in `/docs/features/` before or during implementation.

Each file must include:

- Purpose
- User flow
- Files changed
- Database/API changes
- Security notes
- Testing notes
- Known issues
