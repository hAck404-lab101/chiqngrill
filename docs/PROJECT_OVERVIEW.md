# Project Overview

## Project Name

Chiq-N-Grill Web App

## Business Type

Chicken restaurant and grill house in Accra.

## Known Business Details

- Name: Chiq-N-Grill
- Location: Papa Monrovia Street, Accra
- Phone: 053 361 5069
- Price Range: GH₵50–150 per person
- Services: Dine-in, kerbside pickup, delivery
- Known customer love: well-seasoned chicken, jollof rice, herb butter rice, serene atmosphere, soothing music, setup/vibe

## Main Goal

Create a premium web app that helps customers discover the restaurant, browse meals, place orders, call, get directions, and eventually track orders.

## Design Goal

The app must avoid a generic AI-generated look. It should feel custom-built for a real Accra restaurant using real location details, strong food photography, premium typography, warm lighting, and app-like interactions.

## Conversion Goals

1. Get customers to order.
2. Get customers to call.
3. Get customers to open directions.
4. Get customers to view the menu.
5. Get customers to reserve or visit.

## MVP Scope

- Homepage
- Menu preview
- Food cards
- Cart/order flow planning
- WhatsApp/call fallback
- Google Maps/location section
- Reviews/social proof
- Admin dashboard plan
- Security checklist

## Future Scope

- Customer accounts
- Online payment
- Order tracking
- Reservation management
- Promo codes
- Loyalty points
- Delivery zones
- Kitchen display
- Rider dashboard
