# Chiq-N-Grill Web App

A premium restaurant web app concept for **Chiq-N-Grill**, a chicken and grill restaurant on Papa Monrovia Street, Accra.

This repository starts with documentation-first planning so the app does not become a generic AI-generated restaurant site. The goal is a clean, luxury, app-like ordering experience with strong local context, conversion flow, and production discipline.

## Brand Direction

**Tagline:** Bold Chicken. Smooth Vibes. Accra’s Grill Experience.

**Positioning:** Chiq-N-Grill is a stylish Accra chicken and grill spot for bold flavor, comfort sides, dine-in vibes, pickup, and delivery.

## Core MVP

- Luxury homepage
- Digital menu preview
- App-like ordering CTA flow
- Pickup / delivery / dine-in positioning
- Reviews and social proof
- Location and click-to-call actions
- Admin/order system planning docs
- Security and production checklist docs

## Recommended Tech Direction

- Next.js App Router
- TypeScript
- Tailwind CSS with a custom design system
- Supabase or PostgreSQL later
- Paystack later
- Google Maps embed later
- WhatsApp/click-to-call fallback for ordering

## Project Rules

Before any new feature is added:

1. Create a dedicated branch from `main`.
2. Read the relevant Markdown docs first.
3. Create a feature reference file in `/docs/features/`.
4. Build the feature in small commits.
5. Run code, security, and UI review before merging.
